# MRN - File Information/Structure

*These notes are built off [Ryan's technical breakdown](https://github.com/ryanjsims/warpgate/blob/master/doc/formats/mrn.hexpat).*

MRN files are produced by NaturalMotion's
[Morpheme animation middleware](https://web.archive.org/web/20121013084931/http://www.naturalmotion.com/products/morpheme-3/).

MRN may possibly stand for "Morpheme Rigging Network".

**Extension**: `.mrn`

## Format

**Endianness**: Little
